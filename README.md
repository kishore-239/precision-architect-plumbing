# Precision Architect Plumbing

A website I built for a plumbing business. Has a home page, services, about, and a booking page. Kept it simple — just HTML, CSS, and a bit of JS with Vite for the dev setup.

Live site: https://precision-architect-plumbing-v1.surge.sh

---

To run it locally:

```bash
npm install
npm run dev
```
