# Precision Architect Plumbing

A frontend website project I built for a fictional high-end plumbing business called Precision Architect. The idea was to make a plumbing company look less like a yellow pages listing and more like a serious service brand — clean layout, strong typography, minimal clutter.

Live: https://precision-architect-plumbing-v1.surge.sh

---

## What's inside

Four pages total:

- **Home** — landing page with a hero section and a quick overview of what they offer (leak detection, system redesign, thermal systems)
- **Services** — goes deeper into what the business does, with a bit more visual treatment
- **About** — short company background, been around since 1995, certified technicians, that sort of thing
- **Book** — simple booking/contact page where someone can request an assessment

---

## How it's built

No frameworks, just HTML, CSS, and a small amount of JS. Used Vite mostly for the local dev server and to keep things organized. The styling is all custom — no Tailwind or Bootstrap, wrote the CSS from scratch.

Deployed to Surge since it's a static site and that was the easiest option.

---

## Running locally

```bash
npm install
npm run dev
```

That's it. Opens on localhost:5173 by default.
